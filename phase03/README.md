# Phase 3: DynamoDB レビュー投稿・蓄積

## 概要

Amazon DynamoDB を使って、ユーザーがレビュー（口コミ）を投稿・閲覧できる機能を追加する。

## フォルダ構成

```
phase03/
├── backend/     # Node.js + Express バックエンド
│   └── src/
│       └── routes/
└── frontend/    # React + Vite フロントエンド
    └── src/
        └── pages/
```

## 追加機能

- DynamoDB テーブルへのレビュー書き込み・読み出し
- レビュー投稿フォーム（フロントエンド）
- レビュー一覧表示（フロントエンド）
