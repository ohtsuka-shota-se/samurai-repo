# AWS 学習ハンズオン アプリ

AWS の主要サービスを段階的に学ぶためのハンズオン用アプリです。

## ハンズオン手順書

→ **[docs/ フォルダ](./docs/README.md)** を参照してください。

## フェーズ構成

| フォルダ | 内容 |
|---------|------|
| `phase01/` | EC2 + S3 + IAM（ファイル管理アプリ） |
| `phase02/` | + Cognito（ログイン・認証） |
| `phase03/` | + DynamoDB（レビュー投稿・蓄積） |
| `phase04/` | + Lambda + SNS（レビュー投稿メール通知） |
| `phase05/` | + Bedrock（AI分析・傾向可視化） |
